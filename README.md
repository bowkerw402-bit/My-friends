# Shared Agent Vault

Shared knowledge base between Claude Code and Codex. Both agents read and write here to stay in sync.

## Structure

- `context/` — active project context, goals, decisions
- `claude/` — notes Claude leaves for Codex
- `codex/` — notes Codex leaves for Claude
- `tasks/` — shared task tracking and handoffs
- `log/` — running log of what each agent has done
